\doxysubsection{Detailed Description}
T\+C\+P/\+IP I\+Pv4 server socket. Waits for requests to come in over the network. T\+C\+P/\+IP sockets do not preserve record boundaries but \mbox{\hyperlink{classSocketBuffer}{Socket\+Buffer}} solves this problem.
